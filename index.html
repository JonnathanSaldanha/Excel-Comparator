Configurações de Comparação

Como funciona: Selecione as colunas que deseja comparar em cada arquivo. O sistema irá comparar os conjuntos de valores das colunas selecionadas entre todos os arquivos, independente da ordem das colunas.

Filtro automático: Apenas correspondências com 80% ou mais de similaridade serão consideradas.

⌨️ Atalhos de Teclado:

Ctrl + 1-9 Alternar entre abas de arquivos
Ctrl + Enter Iniciar comparação
Esc Limpar seleção
